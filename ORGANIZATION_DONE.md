## 🚀 حالا چیکار کنم؟

### 1. تست اکستنشن

```bash
# در VS Code
F5
# یا
Run > Start Debugging
```

### 2. ساخت Package

```bash
npx vsce package
# خروجی: multi-language-lorem-with-categories-2.0.0.vsix
```

### 3. Publish (اختیاری)

```bash
# نیاز به Personal Access Token دارید
npx vsce login shahkochaki
npx vsce publish
```
